# Superintendencia de Sociedaddes - Vista 360

# Introdución

> Proyecto de visualización de vista 360.

> URL de acceso [Vista 360 Test](https://vista360.test.supersociedades.htsoft.co/).

# Instalación

```
npm install
```

# Ejecución

```
npm run start
```

# Construcción

```
npm run build
```
